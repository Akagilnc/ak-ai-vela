- **官方 URL**: https://www.sfls.cn/
- **WebFetch 结果**: HTTPS_OK + IS_OFFICIAL ✓
- **位置**: 长宁校区（安顺路 109 号）+ 虹口校区（中山北一路 295 号）

  - 学生在科学 olympiads 有 track record
- **生态社团**: [待 Day 3 确认具体 club 名单]
